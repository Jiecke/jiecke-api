## How To Use❓
![howtouse](https://cdn.jsdelivr.net/gh/mouweng/FigureBed/img/202203292257512.png)
- 1.复制`快捷指令`链接并在iPhone/Mac浏览器中打开
- 2.点击`快捷指令`
- 3.选择机型
- 4.确认选择截图
- 5.从`照片`中选取截图（⚠️必须是截图不能是动态照片）
- 6.等待并完成，程序自动打开`最近照片`
